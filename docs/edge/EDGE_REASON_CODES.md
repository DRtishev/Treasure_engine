# EDGE Reason Codes Registry v1

| code | description |
|---|---|
| NA_REGIME_NOT_SUPPORTED | Regime not supported by this law/config combination. |
| NOT_APPLICABLE | Law is satisfied or not applicable with no error. |
| FAIL_GAPS_POLICY | Missing candle policy violates strict expectation. |
| FAIL_ORDER_DEPENDENCE | Order permutation changed deterministic result. |
| FAIL_CAP_LIMIT | Duplication/scaling encountered bounded-cap behavior. |
| FAIL_FEE_MODEL_SHOCK | Fee shock caused contract breach beyond allowance. |
| FAIL_SPREAD_SPIKE | Spread shock caused contract breach beyond allowance. |
