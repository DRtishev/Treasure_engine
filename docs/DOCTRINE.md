# DOCTRINE

- **Determinism-first**: повторяемые результаты при одинаковых входах.
- **Evidence-first**: PASS подтверждается machine-readable отчетами.
- **Offline-first**: стандартные PASS-гейты не зависят от сети.
- **Safety-first**: live-пути выключены по умолчанию.
- **SSOT discipline**: активные документы должны быть актуальны, противоречия архивируются.
