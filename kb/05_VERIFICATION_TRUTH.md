# Verification Truth

## TruthStack
Core chain: `npm run verify:specs`, `npm run verify:repo`, `npm run verify:ledger`, `npm run verify:treasure`.

## Phoenix and freeze
Phoenix regression chain uses `npm run verify:phoenix` and freeze checks in `npm run verify:epochs:freeze`.

## Evidence and release doctrine
Release integrity requires `npm run release:build` and strict verification with `npm run verify:release`.
