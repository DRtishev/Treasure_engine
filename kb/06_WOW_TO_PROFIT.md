# WOW to Profit

## Acceptance as falsification
Every WOW must declare measurable `must_improve` and `must_not_break` constraints in `specs/wow/WOW_LEDGER.json`.

## Kill criteria discipline
WOW retirement rules and evidence are governed by `specs/wow/WOW_CONTRACTS.md` and `specs/wow/IDEA_CEMETERY.md`.

## Passports as proof
`npm run truth:passports` emits shipped passports to `reports/truth/passports/`; `npm run verify:passports` enforces hash integrity.
