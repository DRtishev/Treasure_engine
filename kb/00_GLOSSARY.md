# Glossary

## Core terms
- **TruthStack**: deterministic verification chain (`npm run verify:specs`, `npm run verify:repo`, `npm run verify:treasure`).
- **Evidence pack**: epoch folder under `reports/evidence/EPOCH-##/` with `pack_index.json` and `SHA256SUMS.EVIDENCE`.
- **Passport**: shipped WOW proof artifact under `reports/truth/passports/`.
